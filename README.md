# Student Management Dashboard

A front-end project built using HTML, CSS, and JavaScript.

## Features
- Add, Edit, Delete student records
- Search functionality
- Course filter
- localStorage data persistence
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- DOM Manipulation
- localStorage

## Project Purpose
This project demonstrates CRUD operations and dynamic UI updates using JavaScript.
